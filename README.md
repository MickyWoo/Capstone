# Capstone Project - Stock API - News API COmbo

## create
```
vue create capstone
vue add vuetify
```
## initial setup
```
vue.config.js 

 and inside 
module.exports = {
  publicPath: '',
  outputDir: 'docs'
}

```

## Project setup
```
npm install
npm install axios
npm install @amcharts/amcharts4
npm install --save vue2-animate

```
