# Capstone Project - Ebay API - Browsing Features

## create
```
vue create capstone
```
## initial setup
```
vue.config.js 

 and inside 
module.exports = {
  publicPath: '',
  outputDir: 'docs'
}

```

## Project setup
```
npm install
npm install axios
npm install vue-chartjs chart.js --save  (for line chart )
```
